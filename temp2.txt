3. FUNCTIONS.PHP

Responsibility:
- Register Book Custom Post Type
- Register Shortcode
- Display Books on frontend

Code:

<?php

defined('ABSPATH') || exit;

function register_book_post_type() {

	register_post_type('book', array(
		'labels' => array(
			'name'          => 'Books',
			'singular_name' => 'Book',
			'add_new'       => 'Add New Book',
			'edit_item'     => 'Edit Book',
		),
		'public'        => true,
		'menu_icon'     => 'dashicons-book',
		'has_archive'   => true,
		'rewrite'       => array('slug' => 'books'),
		'supports'      => array('title', 'editor', 'thumbnail'),
		'show_in_rest'  => true
	));
}
add_action('init', 'register_book_post_type');


function book_list_shortcode() {

	ob_start();

	$query = new WP_Query(array(
		'post_type'      => 'book',
		'posts_per_page' => -1
	));

	if ($query->have_posts()) {

		echo '<div class="book-list">';

		while ($query->have_posts()) {
			$query->the_post();

			echo '<div class="book-item">';
			echo '<h3>' . esc_html(get_the_title()) . '</h3>';
			echo '<div>' . get_the_content() . '</div>';
			echo '</div>';
		}

		echo '</div>';

	} else {
		echo '<p>No books found</p>';
	}

	wp_reset_postdata();

	return ob_get_clean();
}
add_shortcode('book_list', 'book_list_shortcode');


4. FRONTEND USAGE

Shortcode:
[book_list]

Where to use:
- Any Page
- Any Post
- Gutenberg Shortcode block


5. ADMIN PANEL RESULT

After child theme activation:
- "Books" menu appears in admin

- Supports block editor
- Archive available at /books


CLASSIC MINIMALISTIC E-COMMERCE LAYOUT (WIREFRAME + COLORS)
============================================================

1) DESIGN GOALS
---------------
✔ Clean and classic layout  
✔ Minimalistic, no clutter  
✔ Aesthetic spacing, soft typography  
✔ Easy to extend (header, footer, sidebar, product grid)

6) CLASSIC COLOR PALETTE (SUGGESTED)
-------------------------------------
Primary Colors:
- Deep Navy Blue        → #1E293B
- Burgundy / Wine Red   → #7D1128
- Classic Black         → #111827
- Royal Deep Green      → #064E3B

Accent / Highlight:
- Gold / Mustard        → #EAB308
- Soft Peach            → #F5C2C7
- Light Blue Tint       → #93C5FD

Neutral / Background:
- Off White             → #F8FAFC
- Light Gray            → #E5E7EB
- Medium Gray (Text)    → #6B7280

-------------------------------------
EXAMPLE THEME COMBINATION:
-------------------------------------
Primary:        #7D1128 (Burgundy)
Accent:         #EAB308 (Gold)
Text Color:     #1E293B (Dark Navy)
Background:     #F8FAFC (Off White)
Card BG:        #FFFFFF (White)
Borders:        #E5E7EB (Light Gray)

Font Suggestion:
- Playfair Display (Headers - Elegant Serif)
- Inter / Poppins / Lato (Clean Sans-serif body)

7) RESPONSIVE RULES
-------------------
✔ Mobile: Collapsible hamburger menu, 2-column product grid  
✔ Tablet: 3-column grid, fixed header  
✔ Desktop: 4-column grid + sidebar visible  

8) FILE STRUCTURE REFERENCE FOR LAYOUT (MVC VIEW SIDE)
-------------------------------------------------------
app/Views/layouts/app.php        → Header + Footer + Main Wrapper  
app/Views/partials/header.php    → Logo + Nav + Search  
app/Views/partials/footer.php    → Footer Content  
app/Views/product/list.php       → Product Listing (grid)  
app/Views/product/detail.php     → Product Details Page  

-------------------------------------------------------

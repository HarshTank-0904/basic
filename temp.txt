2. Database Tables
-------------------
Table: admins
- id (PK)
- username
- password
- email

Table: categories
- id (PK)
- name
- parent_id (FK to categories.id)  // For subcategories

Table: products
- id (PK)
- category_id (FK)
- name
- description
- price
- stock
- image

Table: users
- id (PK)
- name
- email
- password
- address

Table: cart
- id (PK)
- user_id (FK)
- product_id (FK)
- quantity

Table: orders
- id (PK)
- user_id (FK)
- total_amount
- payment_status
- order_status
- created_at

Table: order_items
- id (PK)
- order_id (FK)
- product_id (FK)
- quantity
- price

Table: settings
- id (PK)
- theme_color
- font_style

3. Implementation Steps / Algorithm
------------------------------------
Admin Side:
1. Admin Login:
   - Display login form.
   - Verify credentials against 'admins' table.
   - Start session after login.

2. Category Management:
   - Form to add new category/subcategory.
   - Update and delete options.
   - Save in 'categories' table.

3. Product Management:
   - Form for adding product with image upload.
   - Edit and delete product.
   - Save in 'products' table.

4. Order Management:
   - Fetch data from 'orders' and 'order_items' table.
   - Change order status: processing, shipped, delivered.

5. Payment Gateway Integration:
   - Use Razorpay, PayPal, Stripe API.
   - Save payment status in 'orders' table.

6. Customization (Color/Font):
   - Form to select theme color and font.
   - Save in 'settings' table.
   - Load dynamically using CSS variables.

User Side:
1. Display Product List:
   - Fetch from 'products' table.
   - Display by category/subcategory.

2. Add to Cart:
   - Add selected product to 'cart' table.
   - Show total items in navbar.

3. Order Summary:
   - Display selected products from cart.
   - Show total price and details.

4. Order Confirmation:
   - Ask user to confirm details.
   - Save in 'orders' table.
   - Redirect to payment gateway.

5. After Payment:
   - Verify payment status.
   - Clear cart table after successful payment.

4. MVC Flow Example (Add to Cart)
-----------------------------------
User clicks "Add to Cart" -> CartController calls model Cart.php -> Inserts data to 'cart' table -> Redirects to cart.php view.

5. Important Notes
---------------------
- Use sessions for login & cart.
- Validate data at server-side.
- Use CSRF token for secure forms.
- Use PDO for secure queries.
- Follow naming conventions.
